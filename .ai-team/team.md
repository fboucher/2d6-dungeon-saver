# Squad Team — Dungeon Saver

## Project Context

**Owner:** fboucher (fboucher@outlook.com)

**Project:** Terminal-based dungeon explorer screensaver that generates procedural dungeons using 2D6 dice mechanics and watches an AI explorer navigate through rooms.

**Tech Stack:** TBD (to be determined by team — likely Python, Rust, or Go for terminal applications)

**Key Features:**
- Full terminal takeover (screensaver-style)
- Procedurally generated dungeons using 2D6 dice roll mechanics
- AI explorer with pathfinding
- Top-down ASCII/Unicode view
- Real-time animation at 10 FPS
- Catppuccin Mocha color theme
- Map export to text files
- Linux terminal compatibility

**Documentation:**
- Rules.md — high-level project features and requirements
- 2D6 Rules.md — dungeon generation mechanics (adapted from pen & paper game)
